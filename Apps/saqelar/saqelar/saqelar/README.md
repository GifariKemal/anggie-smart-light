The Flutter front end for the Anggie smart light. Dark control room dashboard with a live lux gauge, trends, and a safety panel. Reads real device telemetry over WiFi, with a built in simulator fallback.

## 🚀 Run

```powershell
flutter pub get
flutter run
flutter build apk --release
```
